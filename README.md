# Imot3kTogether

Imot3kTogether est un bot discord pour le serveur communautaire Imot3k Community

## Comment l'installer ?

```bash
git clone https://github.com/Imot3k-Community/imot3ktogether
cd imot3ktogether
npm i
```

## Comment le configurer ?

```json
//config.json
{
  "token": "yourtoken"
}
```
